
Блок расширения памяти БРП-4
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Программы из блока расширения памяти "Электроника БРП-4",
предназначенного для эксплуатации совместно с советским
программируемым микрокалькулятором МК-52.

Руководство по эксплуатации перенабрано Black_queen152,
содержимое блока предоставлено и проверено apgor.


Назначение и технические характеристики БРП-4
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Блок предназначен для постоянного хранения пакета прикладных
программ медико-профилактического, бытового и игрового
характера, выполняемых с помощью микрокалькулятора.
Медико-профилактические программы используются для вычисления
ряда физиологических показателей, получения диетических
рекомендаций, оценки физической подготовленности и
работоспособности человека по научно обоснованным тестам.
Бытовые программы позволяют решать задачи исчисления
подоходного налога, определять варианты размена и раздела суммы
денег, переводить метрические единицы в неметрические (единицы,
применяемые в англоязычных странах, и/старые русские) и
наоборот. Игровые программы охватывают наиболее
распространенные числовые и комбинационные игры.

Все программы блока БРП-4 в одном файле:
http://mk.semico.ru/prog/mk/brp4/brp4.tar.gz - архив TAR.GZ;
http://mk.semico.ru/prog/mk/brp4/brp4.zip    - архив ZIP.


Медико-профилактические программы
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Питание - калорийность, потребность в минеральных веществах и витаминах
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
brp4-1.mkl  Расчет рационального питания людей, занятых в сфере производства.
brp4-1.mkl  Расчёт рационального питания детей.
brp4-3.mkl  Расчет рационального питания пожилых людей.
brp4-4.mkl  Расчет суточной потребности человека в витаминах А, В1, В2, С, РР.
brp4-5.mkl  Расчет суточной потребности человека в основных минеральных
            веществах.
brp4-6.mkl  Расчёт суточной калорийности.

Здоровье - антропометрия, биоритмы, определение физической работоспособности
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
brp4-7.mkl  Вычисление идеальных антропометрических характеристик тела
            мужчины.
brp4-8.mkl  Вычисление идеальных антропометрических характеристик тела
            женщины.
brp4-9.mkl  Вычисление индекса массы и артериального давления человека.
brp4-10.mkl Вычисление биологических ритмов человека.
brp4-11.mkl Определение физической работоспособности человека по Гарвардскому
            степ-тесту.
brp4-12.mkl Определение степени физической подготовленности человека по тесту
            Купера.
brp4-13.mkl Определение физической работоспособности человека с помощью пробы
            Руфье-Диксона.
brp4-14.mkl Вычисление количества дней до любой даты ХХ и ХХI столетия.

            более иждивенцев.

            работающих по совместительству.

            Отечественной войны, работающих по совместительству.

            одиноких и малосемейных граждан по месту основной работы.
